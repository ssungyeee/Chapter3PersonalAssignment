### Chapter3PersonalAssignment

Chapter 3-1 Unity 게임개발 입문 개인과제  
제출자 유니티 2기 이규성입니다.

#### 과제 개요

1. Unity 를 이용해 게더를 모방해 만드는 과제입니다.
2. 타일맵을 이용해 배경을 꾸밉니다.
3. 기본 UI 들을 활용해 적용하는 연습이 포함됩니다.

필수 요구사항  
1. 캐릭터 만들기
2. 캐릭터 이동
3. 방 만들기
4. 카메라 따라가기

구현에 성공한 선택 요구사항
1. 캐릭터 애니메이션 추가
2. 현재 시간 표시
